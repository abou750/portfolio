SAÉ S101 – Création de reporting à partir d’un SGBD

BUT Science des Données – Semestre 1

Année : 2022–2023
Outils : PostgreSQL, SQL, Metabase
Type : Projet académique

Contexte du projet

Ce projet a été réalisé dans le cadre du BUT 1 Science des Données. L’objectif était de concevoir un environnement de reporting à partir d’une base de données relationnelle contenant des résultats académiques.

Objectifs

  • Interroger une base de données avec SQL
  • Produire des indicateurs statistiques pertinents
  • Créer des tableaux de bord compréhensibles

Mon rôle et démarche

J’ai installé et configuré PostgreSQL, importé les données puis rédigé des requêtes SQL permettant de calculer des moyennes, écarts-types et répartitions par matière. Ces requêtes ont ensuite été intégrées dans Metabase afin de créer des visualisations interactives.
